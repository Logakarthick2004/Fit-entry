# 🚀 FitEntry

### Smart Gym Management Platform

FitEntry is a modern Gym Management SaaS platform designed to help gym owners, trainers, and fitness centers manage members, attendance, memberships, payments, and operations efficiently.

## 🌐 Live Demo

**Website:** https://fit-entry.vercel.app

---

## ✨ Features

### 👥 Member Management

* Add Members
* Edit Members
* Delete Members
* Member Profiles
* Membership Tracking

### ⏰ Attendance Management

* Daily Attendance
* Manual Check-In / Check-Out
* Attendance Reports
* QR Attendance (Upcoming)

* Membership Plans
* Membership Renewal
* Payment Records
* Invoice Generation
* Revenue Tracking

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/your-username/fit-entry.git
cd fit-entry
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build Production

```bash
npm run build
```
